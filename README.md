Console-Based To-Do List App

A simple beginner-friendly To-Do List application made using HTML and JavaScript.  
This project runs in the browser console and allows users to add, view, and delete tasks.

Features

- Add a new task
- View all tasks
- Delete a task
- Quit the app
- Uses JavaScript prompt and console

Technologies Used

- HTML
- JavaScript

 How to Run

1. Download or clone this repository.
2. Open `To-do-list.html` in your browser.
3. Right-click and choose "Inspect".
4. Go to the "Console" tab.
5. Follow the prompts to use the app.

Commands

- `list` - Show all tasks
- `add` - Add a new task
- `delete` - Delete a task
- `quit` - Exit the app
